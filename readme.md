# media-server-api-client

Library to interface with ofertis/media-server

## Installation

Put `ofertis/media-server-api-client` in composer.json along with:.

```php
"repositories": [
    {
        "type": "vcs",
        "url": "http://104.155.46.144/ofertis/media-server-api-client.git"
    }
]
```
